How to use the Cardboard Computer

There are two versions of the Cardboard Computer available: the basic version has just the D and C scales, which allow multiplication and division, and is suitable for beginners and for pre-secondard-school students; the advanced version adds the CI, A, and K scales for three-step multiplication, square roots, and cube roots, and is suitable for secondard-school and post-secondary students.

2. A couple of notes before we start ðŸ”—

2.1. Orders of magnitude ðŸ”—

Each "trip around the wheel" for the D and C (and CI) scales represents an order of magnitude. That means that 2.1 can also represent 21, 210, 2,100, or (in the other direction) .21 or .021. You need to consider the numbers you're multiplying or dividing to decide which order of magnitude applies, as you'll see in the examples below.

2.2. Precision ðŸ”—

Many answers on the Cardboard Computer will be approximate. The design goal is three significant digits, so you can distinguish between 490 and 491 or 49.0 and 49.1, but not necessarily between 4,900 and 4,901.

Precision-engineered commercial slide rules were often accurate up to 4 or 5 significant digits for experienced users. You might sometimes be able to achieve that level of precision with your own Cardboard Computer, depending on how large you make it and how carefully you align the two wheels, but regardless, you won't get the long sequence of apparently exact digits that you will with a computerised calculator. That's not necessarily a problem: many classic aircraft and even early rockets were designed mostly using analogue slide rules.

3. Multiplication and division ðŸ”—

The D and C scales are spaced so that at any point, the number on the D scale and the number immediately below on the C scale are in the same proportion: for example, if 4 appears above 2, then 6 will appear above 3 and 8 will appear above 4. This relationship allows for basic multiplication and division by putting one of the numbers above 1 (the unit pointer, with a black background).

3.1. Multiplying two numbers ðŸ”—

  1. Find the first number on the D scale and rotate so that it appears above the black unit pointer (1) on the C scale.
  2. Preserving the position of the wheels, find the second number on the C scale
  3. Read the result directly above on the D scale (you can use the cursor, if you added one, to help line up the two numbers.

Example: 41 Ã— 12 = ?

  1. Rotate the wheels until 4.1 (representing 41) on the outer D scale is directly above the unit pointer (1) on the inner C scale.
  2. Without turning the wheels, find 1.2 (representing 12) on the C scale.
  3. Read the number 4.92 (representing 492) on the D scale directly above 1.2 on the C scale. Use the cursor to help, if you added one.

3.2. Dividing two numbers ðŸ”—

  1. Find the first number on the D scale.
  2. Rotate the wheels until the second number appears on the C scale directly underneath the first number.
  3. Preserving the position of the wheels, read the answer on the D scale above the unit pointer (1) on the C scale (you can use the cursor, if you added one, to help line up the answer).

Example: 15 Ã· 4

  1. Find 1.5 (representing 15) on the outer D scale.
  2. Rotate the wheels until 4 on the inner C scale is directly beneath 1.5.
  3. Without turning the wheels, read the answer 3.75 on the D scale directly above the unit pointer (1).
